💖 Study Quiz — Setup Guide

Let's get the quiz game running on your computer. We'll go one little step at a time.

✨ No experience needed — about 15 minutes
0 of 6 steps doneTap a step to open it 👇

🧺 What you'll need first

  • A Windows computer
  • Minecraft (the normal Bedrock version you play)
  • About 15 minutes ☕

Good to know: "Study Quiz" runs on a tiny game server — that's just a small free program from Minecraft that hosts your world. Don't worry, we'll set it up together.

1

Get the game server

▶

First we download the little server program from Minecraft. It's free.

⬇️ Open the Minecraft server download page

On that page, check the box and download the Windows version. Then:

  • Make a new folder somewhere easy, like on your Desktop, named MinecraftServer.
  • Unzip the file you downloaded into that folder (right-click the downloaded file → Extract All).

You'll know it worked if you see a file called bedrock_server.exe inside that folder.

2

Download Study Quiz

▶

Now grab the Study Quiz files.

⬇️ Open the Study Quiz download page

Under Assets, click StudyQuiz-Full-Project.zip to download it. Then unzip it (right-click → Extract All) somewhere easy to find.

3

Run the installer

▶

This copies Study Quiz into your server for you.

  • In the Study Quiz folder you just unzipped, find install-bds.bat and double-click it.
  • A black window opens and asks for your server folder. Paste the path to the MinecraftServer folder from Step 1 and press Enter.

💡 To get the folder path: open the MinecraftServer folder, click the address bar at the top, and copy what's there.

If Windows shows a blue "Windows protected your PC" box, that's normal for this kind of file. Click More info → Run anyway. It's a tiny text file you can open and read.
4

Turn Study Quiz on in your world

▶

One little step the installer can't do for you: tell your world to use Study Quiz. You'll make two small text files.

Go into your server folder → open the worlds folder → open the folder inside it (usually called Bedrock level). Inside that folder, create these two files:

File 1 — name it exactly: world_behavior_packs.json

[{"pack_id":"7f01af09-a5e4-45cf-9f36-696f96a50c0b","version":[1,1,0]}]

File 2 — name it exactly: world_resource_packs.json

[{"pack_id":"b2d6f3a1-9c44-4e7a-8f12-3a7e5c901d44","version":[1,0,0]}]
Making a text file: right-click in the folder → New → Text Document, paste the text, then save it with the exact name above. Tip: turn on View → File name extensions in the folder window so the name doesn't secretly end in .txt.
5

Start it up

▶

In your Study Quiz folder, double-click start-studyquiz.bat. The first time, it'll ask for your server folder again (paste the same one). After that it's one click!

A window opens and gets the server going. Wait until you see this line:

[StudyQuiz] Loaded.

That means it's ready. Leave that window open while you play.

If you're on the same computer and Minecraft can't find the server later, run this once: open PowerShell (search for it in the Start menu) and paste this, then press Enter:
CheckNetIsolation LoopbackExempt -a -n="Microsoft.MinecraftUWP_8wekyb3d8bbwe"
This just lets Minecraft talk to your own computer.
6

Play! 🎉

▶

Open Minecraft on the same computer. Then:

  • Click Play → the Servers tab → scroll down → Add Server.
  • Name: anything you like. Server Address: 127.0.0.1   Port: 19132
  • Save, then Join.

Once you're in, look in your inventory for the Study Settings book 📖 — use it (hold and right-click) to open the quiz menu. Take a quiz, try Lessons, or chat with Ask Claude!

🎉 You did it!

Study Quiz is up and running. Have fun studying — you earned it. 💖

🤖 Want endless AI questions? optional＋

Study Quiz works great with its built-in questions. But if you want endless questions on any subject (and the Curriculum, Lessons, and Ask Claude features), you can connect AI. You'll need:

  • Node.js — a free program. Get it at nodejs.org and install it.
  • Your own AI key from an AI provider (for example, the Anthropic console). This is yours and stays on your computer.

Then, in the Study Quiz folder, open the proxy folder and double-click start-proxy.bat. The first time, it makes a file for your key and opens it — paste your key in, save, and run it again. Start it before the server each time.

After that, the one-click start-studyquiz.bat starts both for you. Your key never leaves your computer.

👫 Want a friend to join? optional＋

Same house / same WiFi: while your server is running, your friend opens Minecraft → Servers → Add Server and uses your computer's address on port 19132.

To find your computer's address, open PowerShell and type ipconfig — look for the "IPv4 Address" that starts with 192.168.

Far away (different house): the easiest safe way is a free tool called Tailscale — you both install it, and your friend connects to the address it gives you. No complicated router settings.

Keep your computer on and the server window open while your friend plays — the world lives on your computer.
🛟 Something not working?＋
The coin or items look like purple-and-black squares.Your world isn't using the resource pack. Double-check Step 4 (the two files are named exactly right and in the world folder), then restart the server.
The Study Settings book isn't in my inventory.Make sure the server window showed [StudyQuiz] Loaded. If not, re-check Step 4 — the version numbers in the files must match (behavior pack is [1,1,0]). An operator can also type /scriptevent study:open in chat to open the menu.
Minecraft won't connect to my own computer.Run the PowerShell line from Step 5 (the CheckNetIsolation one), then try joining again.
Quizzes feel basic or repeat.That's the built-in question set. Turn on AI (the 🤖 section above) for endless fresh questions.
It says "could not make an HTTP request" when using AI.The AI helper isn't running. Start start-proxy.bat before the server, and make sure your key is pasted in.
